# Dancing Monkey AI

A versatile AI voice agent with custom knowledge base support.

## Run Locally

1. `npm install`
2. Create `.env.local`: `GEMINI_API_KEY=your_key_here`
3. `npm run dev`

## Deploy Free on Vercel

1. Push to GitHub
2. Sign up free at vercel.com
3. Import repo, add env var `GEMINI_API_KEY`
4. Deploy - free forever
